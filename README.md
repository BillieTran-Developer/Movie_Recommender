## Movie_Recommender

**Description:**
<br><br>
Through the use of a movie title, the movie recommender can generate a list of movies based on clusters generated by movie rankings and the number of votes. 

**Built With**
- Python
- Javascript
- Flask
- Matplotlib
- Pandas
- Bootstrap
- Etc.
